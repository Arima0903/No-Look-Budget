# No-Look-Budget プレミアム機能構成案

アプリの収益化を見据え、無料版（基本のNo-Look Experience）の体験を損なわず、より高度な家計管理や分析、カスタマイズを求めるユーザー向けの「プレミアム機能（サブスクリプション）」として以下の機能を検討・実装予定としています。

## 1. 借金の分割・返済プランニング機能（Monthly Review機能拡張）

無料版では月末に発生した赤字（借金）は翌月のベース予算から一括で差し引かれますが、高額な出費が重なった場合、翌月の予算が極端に少なくなり継続のモチベーションを下げる原因となります。
プレミアム版では、この赤字を「複数月に分割して返済するプラン」を構築できる機能を提供します。

* **機能詳細:** 月末の振り返り画面にて、赤字額を「翌月一括」か「数ヶ月リカバリプラン」から選択可能にする。
* **効果:** ユーザーのモチベーション低下を防ぎ、長期的なアプリ利用（LTV向上）に直結する。

## 2. カテゴリ上限の解放と高度なカスタマイズ

無料版では認知負荷を下げるためカテゴリ表示上限を「6個」に制限していますが、より細かく管理したいユーザー向けに上限をアンロックします。

* **機能詳細:** カテゴリ作成上限（6個）の解除。カスタムカラーパレットの適用、カスタムアイコンの適用。
* **効果:** 家計簿アプリのヘビーユーザー層のニーズを満たす。

## 3. 過去データへの無制限アクセスと高度な分析レポート

無料版では直近数ヶ月（例: 3ヶ月）のデータ閲覧のみとし、過去の長期的なデータ推移の確認をプレミアム化します。

* **機能詳細:** 全期間のトランザクション履歴・振り返り結果の閲覧。グラフ表示（棒グラフ、トレンドライン）による支出傾向の分析レポート出力機能。
* **効果:** 「自分の過去の頑張りを見返したい」という資産価値（データロックイン）を高め、解約率を下げる。

## 4. 複数人（家族・パートナー）での閲覧・共有機能

同棲カップルや夫婦など、共同で予算を管理したいニーズに対応します。

* **機能詳細:** 読み取り専用の共有リンク発行、あるいはiCloudシームレス共有によるリアルタイムな複数人同期。立替メモ機能の拡張（誰が誰に立替えたかの精算機能）。
* **効果:** 共有による口コミ効果（招待）と、複数人での利用による解約ハードルの上昇。

## 5. ホーム画面ウィジェットの高度なカスタマイズ

iOS 18 の WidgetKit を活用したウィジェットについて、プレミアム専用のレイアウトや情報量が多いウィジェットを提供します。

* **機能詳細:** クイック入力付きウィジェットボタンの自由配置、ロック画面ウィジェットの詳細情報表示など。
* **効果:** OSの最新機能をフル活用した体験によるプレミアム感の演出。

---

*※ 現在ソースコード上には `isPremiumEnabled` （AppStorage）フラグを仕込んでおり、Monthly Review の「借金分割機能」等は、このフラグが true の場合のみUIが解放される仕組みとして先行実装しています。*
