# No-Look-Budget 有償プラン機能提案書

> バージョン: 1.0
> 作成日: 2026-03-17
> 対象フェーズ: Phase 2（ユーザー定着後のマネタイズ導入）

---

## 1. 概要

本ドキュメントは、No-Look-Budget の収益化に向けた有償プラン（Premium）の機能構成と価格設定を定義します。
基本方針は「無料版の核心体験（No-Look Experience）を損なわず、ヘビーユーザーや節約意識の高いユーザーへ追加価値を届ける」ことです。

---

## 2. 目玉機能: AIファイナンシャルプランナー

### 2.1 機能概要

ユーザーの手取り額・固定費を入力するだけで、FP（ファイナンシャルプランナー）の知見に基づいた変動費カテゴリごとの月間予算を自動生成する機能です。

- **使用モデル**: `claude-haiku-4-5-20251001`（Anthropic Claude API）
  - 低コスト・高速応答・日本語性能に優れており、家計管理用途に最適
  - 1リクエストあたりの推定コスト: 約0.5〜1円（Input: 数百トークン、Output: 数百トークン）

### 2.2 入力項目

| 項目 | 例 |
|------|-----|
| 月間手取り額 | 250,000円 |
| 固定費（家賃） | 70,000円 |
| 固定費（通信費） | 5,000円 |
| 固定費（保険・サブスク等） | 10,000円 |
| ユーザー要望プリセット | 「貯金重視」など |

### 2.3 ユーザー要望プリセット（7種類）

プリセットを選ぶと、変動費の配分ロジックが自動調整されます。

#### プリセット定義と配分ロジック

**前提**: 手取り250,000円、固定費85,000円 → 可処分額 = 165,000円

| プリセット名 | 貯蓄率 | 食費 | 交際費 | 趣味・娯楽 | 日用品 | 医療・美容 | 予備費 |
|------------|--------|------|--------|------------|--------|-----------|--------|
| 🛡 節約モード | 40%（66,000円貯蓄） | 25% | 5% | 5% | 8% | 5% | 12% |
| 💰 貯金重視 | 30%（49,500円貯蓄） | 27% | 8% | 8% | 8% | 7% | 12% |
| ⚖ バランス型 | 20%（33,000円貯蓄） | 28% | 12% | 12% | 9% | 7% | 12% |
| 🎉 遊び重視 | 10%（16,500円貯蓄） | 25% | 18% | 20% | 8% | 7% | 12% |
| 🍽 グルメ優先 | 15%（24,750円貯蓄） | 38% | 12% | 10% | 8% | 7% | 10% |
| 💄 自己投資 | 15%（24,750円貯蓄） | 25% | 10% | 8% | 8% | 22% | 12% |
| 🏃 健康優先 | 20%（33,000円貯蓄） | 30% | 8% | 5% | 9% | 16% | 12% |

> 注: 上表の%は可処分額（固定費・貯蓄を除いた残額）に対する比率です。

#### プリセット別の配分ロジック詳細

```
節約モード:
  - FP標準の「収入の20〜25%貯蓄」を超える積極的な貯蓄目標
  - 交際費・娯楽を最小化し、食費も自炊前提の予算設定
  - 「まず貯める、残りで生活する」ペイ・ユアセルフ・ファースト思想

貯金重視:
  - FP推奨の「手取りの20〜25%貯蓄」に準じた標準的な資産形成プラン
  - 社会人2〜3年目の資産形成期に適した配分

バランス型:
  - 節約と生活の質のバランス。FPが最もよく薦める「堅実プラン」
  - 継続しやすい無理のない設計

遊び重視:
  - 交際費・娯楽を厚く確保。貯蓄は最低限のリスクヘッジ分のみ
  - 20代前半や単身者向け

グルメ優先:
  - 食費に大きく配分。外食・質の高い食材を楽しむ設計
  - ランチ代・食料品費が高い都市部在住者向け

自己投資:
  - ジム・スキンケア・ファッションなど自分磨きにかける設計
  - 営業職・接客業など見た目が影響する職種向け

健康優先:
  - 医療費・スポーツ用品・サプリ・ジム代を優先
  - 健康意識が高い30代以上向け
```

### 2.4 Claude APIへのプロンプト設計（実装方針）

```
システムプロンプト（概要）:
「あなたはFP2級相当の知識を持つ家計アドバイザーです。
ユーザーの手取り・固定費・ライフスタイルプリセットをもとに、
月間の変動費カテゴリごとの推奨予算額（円）をJSON形式で返してください。
カテゴリは [食費, 交際費, 趣味・娯楽, 日用品, 医療・美容, 予備費] の6種類です。」

ユーザープロンプト:
「手取り: {月間手取り}円
固定費合計: {固定費合計}円
プリセット: {選択プリセット名}
以上の条件で各カテゴリの月間予算を算出してください。」
```

### 2.5 ユーザー体験フロー

```
1. 「AIプランナー」ボタンをタップ（プレミアム限定）
2. 手取り額・固定費を独自テンキーで入力
3. プリセット7種類からスワイプして選択
4. 「プランを生成」→ Claude APIに送信（数秒）
5. カテゴリ別予算案を確認画面で表示
6. 「この予算でスタート」で全カテゴリに一括反映
7. 後から個別調整も可能
```

## 3. 追加の有償プラン機能案

### 3.1 赤字の分割リカバリープラン（実装済みフラグあり）

- **概要**: 月末に発生した赤字を翌月一括ではなく2〜6ヶ月に分割して繰り越す機能
- **対象ユーザー**: 高額出費（急な冠婚葬祭・旅行等）で予算を大きく超えてしまったユーザー
- **実装状況**: `isPremiumEnabled` フラグで制御済み（解放待ち）
- **価値**: モチベーション維持によるLTV向上。翌月予算が異常に少なくなって「もう無理」と脱落するユーザーを防ぐ

### 3.2 カテゴリ無制限アンロック＋カスタムデザイン

- **概要**: 無料版6カテゴリ上限を解除し、アイコン・カラーをカスタマイズ可能に
- **対象ユーザー**: 家計管理にこだわりがあるヘビーユーザー
- **機能詳細**:
  - カテゴリ数: 6個 → 無制限
  - カスタムアイコン（SF Symbols全種から選択）
  - カスタムカラー（Astronautテーマ準拠のパレット）
  - ウィジェット表示カテゴリの選択（優先度ソート）

### 3.3 過去データ無制限閲覧＋高度な分析レポート

- **概要**: 無料版は直近3ヶ月のみ閲覧可。プレミアムでは全期間アクセス可能
- **機能詳細**:
  - 全期間のカレンダービュー
  - 月別・カテゴリ別トレンドグラフ（折れ線・棒グラフ）
  - 年間サマリーレポート（PDF出力）
  - 「節約できた月ランキング」などのゲーミフィケーション要素
- **価値**: データロックイン効果で解約率を抑制

### 3.4 ウィジェット高度カスタマイズ

- **概要**: ウィジェットのレイアウト・表示内容をより細かく設定可能に
- **機能詳細**:
  - ウィジェットに表示するカテゴリを最大3個まで任意選択（現在は固定表示）
  - ダークテーマのカスタムカラーパレット（5種類のプリセット）
  - ロック画面ウィジェットの文字サイズ変更
  - 残高パーセンテージ表示 / 絶対額表示の切替

### 3.5 IOUプレミアム（立替管理の強化）

- **概要**: 立替記録の高度な管理機能
- **機能詳細**:
  - 立替相手（名前・連絡先）の登録とグループ管理
  - LINE/メッセージへの「請求文テキスト」ワンタップ送信
  - 立替の期日設定とリマインダー通知
  - 立替精算履歴の統計（誰が一番払ってくれないか等）

### 3.6 スマート通知・アラート設定

- **概要**: 行動経済学に基づいた支出抑制通知のカスタマイズ
- **機能詳細**:
  - カテゴリ別残高〇%以下でプッシュ通知（閾値を自由設定）
  - 「今日まだ入力していない」リマインダー（時刻指定）
  - 週次サマリー通知（月曜朝に先週の振り返り）
  - 「あと〇日で月末。現在ペースで行くと△△円オーバー」の予測通知

### 3.7 iCloud同期・マルチデバイス対応

- **概要**: SwiftDataのデータをiCloudKeychainで暗号化同期
- **対象ユーザー**: iPhone/iPad両持ち、機種変更が不安なユーザー
- **機能詳細**:
  - iCloudを介した自動バックアップ・復元
  - 機種変更時のデータ引き継ぎ保証
  - 無料版はローカル保存のみ（機種変更でデータ消失リスクを訴求）

### 3.8 行動経済学モード（チャレンジ機能）

- **概要**: ゲーミフィケーション要素で節約をモチベーション化
- **機能詳細**:
  - 「今月交際費を1万円以内」などの自己宣言チャレンジ
  - 達成時のAstronaut演出（宇宙船打ち上げアニメーション等）
  - 連続達成日数ストリーク表示（Duolingo式）
  - 将来オプション: 予算オーバー時の「罰金積立」（別口座への自動送金ショートカット連携）

## 4. 価格設定

### 4.1 競合アプリとの機能・価格比較

| アプリ名 | 月額（円） | 年額（円） | 主な対象 | AI機能 | ウィジェット |
|---------|----------|----------|---------|--------|------------|
| マネーフォワード ME プレミアム | 500円 | 5,300円 | 几帳面な管理志向 | なし | 限定的 |
| Zaim プレミアム | 480円 | 4,800円 | データ分析志向 | なし | なし |
| Moneytree GROW | 500円 | 5,500円 | 資産管理全般 | なし | なし |
| **No-Look-Budget Premium** | **480円** | **4,320円** | **浪費家・ADHD気質** | **あり（AI予算設定）** | **高度カスタマイズ** |

### 4.2 機能価値の定量評価

| 機能 | 定量的価値 |
|------|-----------|
| AIファイナンシャルプランナー | FP相談料 = 1回5,000〜15,000円相当を月1回無料で提供 |
| iCloud同期 | データ消失リスク回避 = 数年分の家計記録の価値 |
| 分割リカバリープラン | 継続率向上 → LTV換算で数千円〜数万円相当 |
| スマート通知 | 浪費防止効果 = 月数千〜数万円の節約効果 |

### 4.3 推奨価格（円）

```
月額プラン: 480円/月
年額プラン: 4,320円/年（月換算360円、25%オフ）
買い切りプラン: 将来検討（5,800円程度）
```

**価格設定根拠**:
- 競合アプリ（マネーフォワード: 500円、Zaim: 480円）と同水準
- AI機能という差別化要素があるため値下げ競争には参加しない
- 年額プランを25%オフに設定し、長期継続を促進
- 「月480円 = コーヒー2杯」という比較で心理的ハードルを下げる

### 4.4 無料/有料の機能境界線

| 機能 | 無料版 | 有償プレミアム |
|------|--------|-------------|
| No-Look Experienceウィジェット | ✅ 全機能 | ✅ + 高度カスタマイズ |
| QuickInput（手動入力） | ✅ 全機能 | ✅ |
| カテゴリ数 | ✅ 最大6個 | ✅ 無制限 |
| 立替セパレーター | ✅ 基本機能 | ✅ + 相手管理・請求送信 |
| カレンダービュー | ✅ 直近3ヶ月 | ✅ 全期間 |
| 月次振り返り | ✅ 基本 | ✅ + 分割リカバリー |
| 分析レポート | ❌ | ✅ |
| AIファイナンシャルプランナー | ❌ | ✅ |
| iCloud同期 | ❌ | ✅ |
| スマート通知カスタマイズ | 基本のみ | ✅ 全設定 |
| チャレンジ機能 | ❌ | ✅ |
| 広告 | あり（将来） | なし |

**境界線設計の思想**:
- 無料版は「中毒になる体験」を完全提供。課金ブロックで体験を壊さない
- 有料版は「もっと便利に・もっと長く使いたい人」への追加価値
- AI機能・iCloud同期・データ無制限の3点が有料版の核心訴求
