# Orbit Budget 有償プラン機能提案書

> バージョン: 2.0
> 更新日: 2026-03-17
> 対象フェーズ: Phase 2（ユーザー定着後のマネタイズ導入）

---

## 1. 基本方針

「無料版の核心体験（No-Look Experience）を損なわず、データとモチベーションを長期的に守りたいユーザーへ追加価値を届ける」

- **無料版**: 中毒になる体験を完全提供。課金ブロックで体験を壊さない
- **有料版**: データ保護・継続サポートの2軸に絞って課金を正当化する
- **廃止方針**: 費用対効果が薄い機能（AI FP・ウィジェットカスタマイズ・IOU強化）は削除してシンプルに保つ

---

## 2. 無償プランで提供する新機能

ユーザー獲得・継続率向上・バイラル促進を目的に、以下を**無料**で提供する。

### 2.1 ライフスタイルプリセット（予算テンプレート）

手取り額と固定費を入力してプリセットを選ぶだけで、各カテゴリの月間予算を自動配分する機能。
AIは使わず、FPの知見に基づくシンプルな配分表をアプリ内に内蔵する。

#### プリセット7種類と配分ロジック

前提例: 手取り250,000円、固定費85,000円 → 可処分額 = 165,000円

| プリセット | 貯蓄率 | 食費 | 交際費 | 趣味・娯楽 | 日用品 | 医療・美容 | 予備費 |
| ---------- | ------ | ---- | ------ | ---------- | ------ | ---------- | ------ |
| 🛡 節約モード | 40% | 25% | 5% | 5% | 8% | 5% | 12% |
| 💰 貯金重視 | 30% | 27% | 8% | 8% | 8% | 7% | 12% |
| ⚖ バランス型 | 20% | 28% | 12% | 12% | 9% | 7% | 12% |
| 🎉 遊び重視 | 10% | 25% | 18% | 20% | 8% | 7% | 12% |
| 🍽 グルメ優先 | 15% | 38% | 12% | 10% | 8% | 7% | 10% |
| 💄 自己投資 | 15% | 25% | 10% | 8% | 8% | 22% | 12% |
| 🏃 健康優先 | 20% | 30% | 8% | 5% | 9% | 16% | 12% |

> 注: %は可処分額（固定費・貯蓄を除いた残額）に対する比率。

#### ユーザー体験フロー

```text
1. 予算設定画面から「プリセットで設定」をタップ
2. 手取り額・固定費を独自テンキーで入力（済みならスキップ）
3. プリセット7種類をスワイプして選択
4. 「この配分でスタート」→ 全カテゴリに一括反映
5. 後から個別調整も可能
```

提供理由: 「最初の予算設定が面倒で脱落する」ユーザーへの導線。無料で提供することでオンボーディング完了率を高める。

---

### 2.2 スマート通知・アラート設定

行動経済学に基づいた支出抑制通知。設定の自由度を無料で全開放する。

- カテゴリ別残高が任意%以下になったらプッシュ通知（例: 食費残高20%以下で通知）
- 「今日まだ入力していない」リマインダー（時刻指定）
- 週次サマリー通知（月曜朝に先週の振り返り数値を送付）
- 予測通知「あと〇日で月末。現在ペースで行くと△△円オーバー」

提供理由: 通知をトリガーにアプリを開く習慣が生まれ、継続率が上がる。課金機能にするには価値が小さすぎる。

---

### 2.3 行動経済学モード（チャレンジ機能）

ゲーミフィケーションで節約をモチベーション化し、SNSシェアでバイラルを狙う。

#### 月間チャレンジ（自己設定型）

- ユーザー自身が宣言：「今月の交際費は1万円以内」「食費を先月比-3,000円」
- 月初にチャレンジを設定し、月末に自動判定

#### 達成演出（Astronautテーマ）

- 達成時: 宇宙船打ち上げアニメーション + 「MISSION CLEARED!」演出
- 達成できなかった時: 「次のMISSIONへ。宇宙飛行士は諦めない」などの励ましメッセージ

#### 連続達成ストリーク

- Duolingo式の連続達成日数カウンター
- 3ヶ月連続達成で「ベテラン宇宙飛行士」バッジなどの称号付与

#### 節約額の可視化

- 「今月、交際費チャレンジで4,200円節約！ ☕️コーヒー約14杯分」などに換算表示
- 「節約した金額の累計」を宇宙旅行の距離に換算（遊び心のある表示）

#### SNSシェア機能（バイラル促進）

- 「今月食費チャレンジ達成！ ¥8,400節約しました🚀 #OrbitBudget」
- Astronautテーマのシェア画像を自動生成（宇宙飛行士と達成額が入ったカード）

提供理由（無料化の戦略的意図）:

- 達成時のシェアが自然な口コミになる
- チャレンジ継続のためにアプリを開く習慣が形成される（D30継続率向上）
- 「節約できた体験」が有料プランへの心理的な壁を下げる

## 3. 有償プラン機能

### 3.1 赤字の分割リカバリープラン【有償確定・実装済みフラグあり】

- **概要**: 月末に発生した赤字を翌月一括ではなく2〜6ヶ月に分割して繰り越す機能
- **対象ユーザー**: 高額出費（急な冠婚葬祭・旅行・家電故障等）で予算を大きく超えたユーザー
- **実装状況**: `isPremiumEnabled` フラグで制御済み（解放待ち）
- **価値**: 翌月予算が「実質ゼロ」になって「もう無理」と脱落するユーザーを防ぐ。継続率向上がLTVに直結

なぜ有料か: 「苦しい月を乗り越えられる」体験そのものが課金の動機になる。困ったタイミングで有料版の価値を最も実感できる機能。

---

### 3.2 カテゴリ無制限アンロック＋カスタムデザイン

- **概要**: 無料版の6カテゴリ上限を解除し、アイコン・カラーをカスタマイズ可能に
- **対象ユーザー**: 家計管理にこだわりがあるヘビーユーザー・カップル・家族での利用
- **機能詳細**:
  - カテゴリ数: 6個 → 12個まで
  - カスタムアイコン（SF Symbols全種から選択）
  - カスタムカラー（Astronautテーマ準拠のパレット）

---

### 3.3 過去データ無制限閲覧＋高度な分析レポート＋CSV出力

#### 過去データ閲覧

- **無料版**: 直近3ヶ月のみ
- **有償版**: 全期間のカレンダービュー・月次振り返り結果の閲覧

#### 分析レポート

- 月別・カテゴリ別トレンドグラフ（折れ線・棒グラフ）
- 年間サマリーレポート
- 「節約できた月ランキング」「最も使ったカテゴリトップ3」などの統計

#### CSV出力（過去データのエクスポート）

有効なシーン（プロダクトアウト的だが使われる場面は明確に存在する）:

| シーン | 詳細 | 使用頻度 |
| ------ | ---- | -------- |
| 確定申告（1〜3月） | 医療費控除・ふるさと納税の仕分けでExcel整理したい | 年1回 |
| FP・税理士への相談 | 「半年分の支出データを持参してください」に即対応 | 不定期 |
| パートナーへの家計報告 | スプレッドシートで共有・議論したい | 月1回 |
| 引越し・転職前の見直し | 現状把握してから新生活の予算を設計したい | ライフイベント時 |

評価: 日常的な利用にはならないが「あって助かった」体験が強い機能。確定申告シーズンに「このアプリのCSV出力が神」という口コミが生まれやすい。3.3のおまけ機能として追加する価値はある。

CSV出力仕様（案）:

```csv
date,category,amount,memo,isIOU,isIncome
2026-03-15,食費,800,ランチ,false,false
2026-03-14,交際費,3200,,false,false
```

価値: データロックイン効果で解約率を抑制。「3年分のデータがこのアプリにある」状態になると解約できなくなる。

---

### 3.4 iCloud同期・マルチデバイス対応【有償確定】

- **概要**: SwiftDataのデータをiCloud経由で暗号化同期・バックアップ
- **対象ユーザー**: iPhone/iPad両持ち・機種変更が不安なユーザー
- **機能詳細**:
  - iCloudを介した自動バックアップ・復元
  - 機種変更時のデータ引き継ぎ保証
  - 無料版はローカル保存のみ（機種変更でデータ消失リスクを訴求）
- **なぜ有料か**: 「3年分の家計記録が消えるかもしれない」という不安を解消する価値は月480円を超える

## 4. 廃止した機能と理由

| 機能 | 廃止理由 |
| ---- | -------- |
| AIファイナンシャルプランナー（旧Section 2） | Claude API利用コストと開発工数に対して使用頻度が低い見込み。プリセット機能（2.1）で代替可能 |
| IOUプレミアム（旧3.5） | 立替管理のヘビーユーザー層が薄く費用対効果が低い |
| ウィジェット高度カスタマイズ（旧3.4） | 費用対効果が薄い。「最大3個まで表示」という仕様記述は誤りで、デフォルトで6カテゴリすべて表示される |

## 5. 価格設定

### 5.1 競合アプリとの機能・価格比較

| アプリ名 | 月額 | 年額 | 主な対象 | ウィジェット |
| -------- | ---- | ---- | -------- | ------------ |
| マネーフォワード ME プレミアム | 500円 | 5,300円 | 几帳面な管理志向 | 限定的 |
| Zaim プレミアム | 480円 | 4,800円 | データ分析志向 | なし |
| Moneytree GROW | 500円 | 5,500円 | 資産管理全般 | なし |
| **Orbit Budget Premium** | **480円** | **4,320円** | **浪費家・ADHD気質** | **フル機能（標準）** |

### 5.2 機能価値の定量評価

| 機能 | 定量的価値 |
| ---- | ---------- |
| iCloud同期 | データ消失リスク回避 = 数年分の家計記録の資産価値 |
| 分割リカバリープラン | 継続率向上 → LTV換算で数千〜数万円相当 |
| 過去データ無制限 + CSV | 確定申告時の医療費整理 = 税理士相談代の節約 |

### 5.3 推奨価格

```text
月額プラン: 480円/月
年額プラン: 4,320円/年（月換算360円、25%オフ）
買い切りプラン: 将来検討（5,800円程度）
```

価格設定根拠:

- 競合アプリ（マネーフォワード: 500円、Zaim: 480円）と同水準
- 「月480円 = コーヒー2杯」という比較で心理的ハードルを下げる
- 年額プランを25%オフに設定し、長期継続を促進
- 有料機能をデータ保護・継続サポートに絞ったことで「保険として払う」納得感が高い

## 6. 機能境界線

| 機能 | 無料版 | 有償プレミアム |
| ---- | ------ | -------------- |
| No-Look Experienceウィジェット（6カテゴリ） | ✅ 全機能 | ✅ |
| QuickInput（手動入力・立替） | ✅ 全機能 | ✅ |
| カテゴリ数 | ✅ 最大6個 | ✅ 最大12個 |
| カレンダービュー・振り返り | ✅ 直近3ヶ月 | ✅ 全期間 |
| ライフスタイルプリセット | ✅ 全7種 | ✅ |
| スマート通知カスタマイズ | ✅ 全設定 | ✅ |
| チャレンジ機能・ストリーク | ✅ 全機能 | ✅ |
| 月次振り返り（基本） | ✅ | ✅ |
| 赤字の分割リカバリープラン | ❌ | ✅ |
| 分析レポート（トレンドグラフ） | ❌ | ✅ |
| CSV出力 | ❌ | ✅ |
| iCloud同期・マルチデバイス | ❌ | ✅ |
| 広告 | あり（将来） | なし |

設計思想:

- 無料版は「使い続けたい」と思わせる体験を完全提供
- 有料版の訴求軸: **データを守る**（iCloud同期）・**継続を守る**（分割リカバリー）・**過去を活かす**（全期間閲覧＋CSV）
